# Olaytech By Application 三层目录页面包

这个包用于先完成 **By Application 的三级结构**：

1. `applications.html`：应用场景总目录
2. `application-*.html`：应用场景分类页，例如 Beauty & Cosmetic、Food & Cooler、Outdoor & Sports
3. `application-*-具体产品.html`：具体产品详情页，带图片区、参数表、Get A Quote、WhatsApp

## 这次需要上传的文件

建议上传：

- `applications.html`
- 所有 `application-*.html`
- `css/application-catalog.css`
- `js/application-catalog.js`
- `assets/products/app-*.jpg`
- `sitemap.xml`
- `robots.txt`

可选上传：

- `assets/brand/olay-logo-black.png`
- `assets/brand/olay-logo-white.png`

## 不需要上传 / 不要覆盖

- 不需要覆盖 `index.html`，主页设计不会被动
- 不需要覆盖 `contact.html`，发邮件功能不会被动
- 不需要覆盖原来的 `css/style.css`

## 图片替换方式

后续你上传真实产品图时，只要覆盖同名图片即可，例如：

- `assets/products/app-beauty-01.jpg`
- `assets/products/app-food-01.jpg`
- `assets/products/app-sports-01.jpg`

页面会自动显示真实图片。
