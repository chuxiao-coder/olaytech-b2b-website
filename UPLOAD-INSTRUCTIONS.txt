Olaytech Navigation Size Lock Files

Upload these files to the same paths in your GitHub website repository:

1) css/homepage-57.css
   Replace the existing css/homepage-57.css

2) js/main.js
   Replace the existing js/main.js

Important:
- Every page must include this line in the <head>:
  <link rel="stylesheet" href="css/homepage-57.css" />

- Every page should include this before </body>:
  <script src="js/main.js"></script>

This package is for fixing navigation size consistency:
- fixed topbar height
- fixed header height
- fixed logo size
- fixed menu font size and spacing
- fixed Get Quote button size
- stable scrollbar gutter to reduce page jump
- mobile menu toggle
