# Olaytech Split Product Pages Upgrade

This package separates product browsing into independent pages:

- products.html: clean product center landing page
- product-types.html: product categories by type
- materials.html: product categories by material
- applications.html: product categories by application

## Product image upload slots

Upload your real product photos into `assets/products/` using the exact file names shown on each card.

Examples:
- assets/products/cosmetic-bags-01.jpg
- assets/products/toiletry-bags-01.jpg
- assets/products/shopping-bags-01.jpg
- assets/products/cooler-bags-01.jpg
- assets/products/canvas-bags-01.jpg
- assets/products/beauty-cosmetic-bags-01.jpg

Before images are uploaded, the website shows a clean placeholder. After images are uploaded with the same file names, the real photos display automatically.
