Olaytech Navigation Only Patch

Only these files are included:
- css/unified-nav.css
- js/main.js
- _redirects

Purpose:
- Redesign and unify the site navigation across pages.
- Normalize logo size, header height, menu text size, dropdown style and Get Quote button.
- Keep old category links redirecting to the new dynamic catalog pages.

This patch does NOT change:
- admin/config.yml
- CMS login or image upload fields
- product data under data/products/
- product images under assets/uploads/
- homepage content blocks
- product detail content fields

How to apply:
1. Unzip this package.
2. Copy the three files into your GitHub local repository, keeping the same folders.
3. Replace existing files when asked.
4. Commit and push to GitHub.
5. Wait for Netlify deploy.
6. Open the site and press Ctrl + F5 to refresh cache.
