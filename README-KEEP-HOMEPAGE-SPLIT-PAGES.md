# Olaytech Split Pages Package - Keep 2.4 Homepage Design

This package preserves the 2.4 INITI-style homepage design and adds separate pages for:

- products.html
- product-types.html
- materials.html
- applications.html

It also keeps the same O'Lay logo and factory image assets.

Upload these files to GitHub and commit changes. The package does not include contact.html, so your Web3Forms email function will not be affected.

Recommended upload list:
- index.html
- products.html
- product-types.html
- materials.html
- applications.html
- css/style.css
- js/main.js
- assets/brand/olay-logo-black.png
- assets/brand/olay-logo-white.png
- assets/factory/aolai-factory-overview.jpg
- sitemap.xml
- robots.txt

Product image placeholders use paths under assets/products/. Upload images with the suggested filenames shown on each card.
