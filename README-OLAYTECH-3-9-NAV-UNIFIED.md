# Olaytech 3.9 Unified Navigation Patch

This package fixes the navigation/header mismatch on About Us, By Application, Support, FAQ, Blog and Contact pages.

## Notes

- Contact form keeps the existing Web3Forms access key and email submission function.
- By Type and By Material three-level catalog pages are not removed or overwritten by this patch.
- Header order is unified as: Home / About Us / By Types / By Materials / By Application / Support / Contact / Get Quote.
