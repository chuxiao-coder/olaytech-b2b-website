# Olaytech 3.7 - By Material Three-Level Catalog

This package keeps the 2.4 homepage design and the three-level By Type catalog, then adds the same three-level structure for By Materials.

## New By Material Structure

```text
materials.html
  ↓
canvas-bags.html / nylon-bags.html / oxford-bags.html / pvc-eva-bags.html / neoprene-bags.html / rpet-bags.html / pu-leather-bags.html / felt-bags.html / non-woven-bags.html
  ↓
material product detail pages, such as canvas-bags-canvas-retail-tote.html
```

## Image Upload Naming

Upload real product photos into:

```text
assets/products/
```

Examples:

```text
canvas-bags-01.jpg
canvas-bags-02.jpg
nylon-bags-01.jpg
pvc-eva-bags-01.jpg
neoprene-bags-01.jpg
```

Do not replace `contact.html` if your Web3Forms inquiry email is already working.
