## How To Use

1. Open `index.html` in your browser to preview.
2. Replace placeholder product cards with your real product images.
3. Upload all files to a GitHub repository.
4. Deploy the repository with Vercel or Cloudflare Pages.
5. Point your domain `www.olaytech.com` to the deployed site.
6. Submit `sitemap.xml` to Google Search Console.

## Important Form Note
