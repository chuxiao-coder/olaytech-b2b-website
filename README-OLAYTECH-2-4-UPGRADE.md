# Olaytech 2.4 Homepage Upgrade

This version follows a B2B bag manufacturer layout inspired by INITI Packing-style structure:

- Navigation organized by product type, material, application and support
- Cleaner hero section without repeating the factory image
- Factory photo appears only once in the About / Factory Partner section
- O'Lay logo is used in the header and footer
- Product Series, Service Capabilities, Premium Service, Why Choose Us, Buyer Guide and Inquiry CTA sections

## Upload to GitHub
Replace/upload only these files:

- index.html
- css/style.css
- js/main.js
- assets/brand/olay-logo-black.png
- assets/brand/olay-logo-white.png
- assets/factory/aolai-factory-overview.jpg
